技术文档
王阳

Hexo简介

A fast, simple & powerful blog framework, powered by Node.js.

工具推荐

  • WebStorm
  • Visual Studio Code
  • MarkdownPad

安装步骤

1. 首先得有Node.js.
2. 其次全局安装hexo
3. git clone 克隆配置完好的Hexo环境

记住一定要获取最新的代码

4. 进入对应目录,安装对应依赖包
5. 创建一篇文章
6. 本地书写文章打开对应目录文件
7. 打开markdown，默认情况

  • title 文章标题
  • date 发布时间
  • tags 标签
  • author 作者
8. markdown语法书写

  • 添加图片统一路径
  • 图片添加的语法
9. 启动Hexo Server 本地查看
10. 发布文章
  • 发布执行请先hexo clean 清除发布历史数据
  • 目前发布文章需要手动输入github用户名和密码进行发布
11. 提交代码，防止其他人搞掉你的文章
